# GaGaUpGameV6
游戏框架, 以数据为驱动, 数据在驱动UI, 并且可以独立于UI运行 多阵营, 多层级(海陆空), 智能AI等

依赖类库 : GaGaFrameClassLibrary_V2
依赖框架 : GaGaFrameForClient_V2

GaGaFrameForDevelopMax
游戏使用的 文件系统, 动画编辑器, 地图编辑器

| **名称**   | **说明**     |
| ---------- | ------------ |
| **engine** | 模型数据处理 |
| **Info**   | 数据模型层   |
| **render** | 渲染控制层   |
| **tools**  | 静态工具层   |
